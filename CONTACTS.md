## Escalation Path

1. **Document-level**: Author → Reviewers → Approver
2. **Process-level**: Documentation Lead → Program Manager
3. **Technical-level**: Domain Expert → Architecture Lead
4. **Policy-level**: Program Manager → Program Director → MoFPI

## Onboarding New Contributors

1. Review AGENTS.md, CLAUDE.md, CONTRIBUTING.md
2. Attend orientation session (monthly)
3. Assigned mentor from domain experts
4. First contribution: supervised iteration
5. Independent after 2 successful iterations
