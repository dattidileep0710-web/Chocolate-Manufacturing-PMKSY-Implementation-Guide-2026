## Usage Rules

1. **First use in each document**: Full term followed by abbreviation in parentheses
   - Example: "Ministry of Food Processing Industries (MoFPI)"
2. **Subsequent uses**: Abbreviation only
3. **Common abbreviations** (PMKSY, MoFPI, FSSAI, BIS): May be used without definition after first use in document
4. **Never**: Define abbreviation differently in different documents
5. **Tables/Figures**: May use abbreviation only if defined in document body

## Adding New Abbreviations

1. Open GitHub Issue with "abbreviations" label
2. Provide abbreviation, expansion, and domain
3. Architecture Lead approves
4. Added to this document (MINOR version bump)
5. All existing documents updated if term used
