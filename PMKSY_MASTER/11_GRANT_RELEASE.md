# Grant Release

## Overview

This document defines the complete PMKSY grant release process for chocolate manufacturing projects in Andhra Pradesh. It covers subsidy calculation, release stages, documentation requirements, bank coordination, agency responsibilities, and common pitfalls.

**OFFICIAL** — PMKSY subsidy is released in installments tied to project milestones. Release is contingent on verified expenditure, physical progress, and compliance with sanction terms.

> **SCOPE BOUNDARY**: This document addresses **grant release mechanics**. It does **not** address:
> - Eligibility criteria (see [PMKS-002: Project Eligibility](PMKSY_MASTER/02_PROJECT_ELIGIBILITY.md))
> - Cost eligibility (see [PMKS-006: Eligible Project Cost](PMKSY_MASTER/06_ELIGIBLE_PROJECT_COST.md))
> - Document requirements (see [PMKS-007: Required Documents](PMKSY_MASTER/07_REQUIRED_DOCUMENTS.md))
> - Application workflow (see [PMKS-008: Application Workflow](PMKSY_MASTER/08_APPLICATION_WORKFLOW.md))
> - Agency roles (see [PMKS-009: Implementing Agencies](PMKSY_MASTER/09_IMPLEMENTING_AGENCIES.md))
> - Appraisal criteria (see [PMKS-010: Project Appraisal](PMKSY_MASTER/10_PROJECT_APPRAISAL.md))

## Grant Release Lifecycle

### High-Level Flow

**PROJECT ASSUMPTION** — The PMKSY grant release lifecycle for a chocolate manufacturing unit:

```
SANCTION → FIRST RELEASE → SECOND RELEASE → FINAL RELEASE → CLOSURE
    ↓           ↓              ↓               ↓            ↓
 Agreement   Expenditure    Expenditure     Expenditure  Project
             certified      certified       certified    completion
             & verified      & verified      & verified
```

### Grant Release Stages

| Stage | Milestone | Typical Timing | Status |
|-------|-----------|----------------|--------|
| **First Release** | After equipment supply / civil completion | 6-12 months post-sanction | PROJECT ASSUMPTION |
| **Second Release** | After installation / partial commissioning | 12-18 months post-sanction | PROJECT ASSUMPTION |
| **Final Release** | After trial production / commercial operation | 18-24 months post-sanction | PROJECT ASSUMPTION |

**REQUIRES RE-VERIFICATION** — Exact milestone definitions and timing vary by implementing agency and project scope.

## Release Stages in Detail

### First Release

#### Purpose
Release initial subsidy installment to support equipment procurement and early civil works.

#### Typical Trigger
**REQUIRES RE-VERIFICATION** — Common triggers for first release:
- Equipment supply order placed and advance paid
- Civil works in progress (foundation, structure)
- **REQUIRES RE-VERIFICATION** — Confirm specific trigger with implementing agency

#### Expenditure Typically Covered
| Category | Status |
|----------|--------|
| Plant and machinery (supply orders) | REQUIRES RE-VERIFICATION |
| Civil works (progress) | REQUIRES RE-VERIFICATION |
| Electrical installations | REQUIRES RE-VERIFICATION |
| Laboratory equipment | REQUIRES RE-VERIFICATION |

#### Typical Release Percentage
**REQUIRES RE-VERIFICATION** — First release typically ranges from 30-50% of total sanctioned subsidy. Confirm with implementing agency.

### Second Release

#### Purpose
Release intermediate subsidy installment to support installation, commissioning, and pre-operative activities.

#### Typical Trigger
**REQUIRES RE-VERIFICATION** — Common triggers:
- Machinery delivery and installation in progress
- Civil works substantially complete
- Pre-operative expenses incurred

#### Expenditure Typically Covered
| Category | Status |
|----------|--------|
| Erection and commissioning | REQUIRES RE-VERIFICATION |
| Electrical and utility completion | REQUIRES RE-VERIFICATION |
| Pre-operative expenses | REQUIRES RE-VERIFICATION |
| Trial production | REQUIRES RE-VERIFICATION |

#### Typical Release Percentage
**REQUIRS RE-VERIFICATION** — Second release typically brings cumulative release to 70-80% of total sanctioned subsidy.

### Final Release

#### Purpose
Release remaining subsidy after project completion and commencement of commercial production.

#### Typical Trigger
**REQUIRES RE-VERIFICATION** — Common triggers:
- Trial production completed successfully
- Commercial production commenced
- All statutory clearances obtained
- Final cost audit completed

#### Expenditure Typically Covered
| Category | Status |
|----------|--------|
| Balance of all eligible costs | REQUIRES RE-VERIFICATION |
| Contingencies (if unspent) | REQUIRES RE-VERIFICATION |
| Overheads (if unspent) | REQUIRES RE-VERIFICATION |

#### Typical Release Percentage
**REQUIRES RE-VERIFICATION** — Final release covers remaining 20-30% of sanctioned subsidy.

### Release Stages Table

| Stage | Trigger | Required Certificates | Responsible Party |
|-------|---------|------------------------|-------------------|
| First | Equipment supply / civil progress | UC, invoices, bank certificate, inspection report | Applicant + Bank + IA |
| Second | Installation / commissioning | UC, invoices, bank certificate, inspection report | Applicant + Bank + IA |
| Final | Commercial production | UC, final cost audit, completion certificate, bank certificate | Applicant + Bank + IA |

## Bank Coordination

### Bank Role in Grant Release

**BANK PRACTICE** — The bank's role is critical because:

| Bank Function | Purpose | Timing |
|---------------|---------|--------|
| Expenditure verification | Confirm applicant has spent claimed amount | Before subsidy release |
| No-due certificate | Confirm no loan overdues | Before subsidy release |
| Loan disbursement | Release applicable loan installment | Coordinated with subsidy release |
| Account monitoring | Track project account transactions | Ongoing |

### Bank-IA Coordination

**BANK PRACTICE** — Bank and Implementing Agency coordinate through:

| Activity | Participants | Frequency |
|----------|--------------|-----------|
| Joint site inspection | Bank officer + IA official | Per grant release |
| Expenditure certification | Bank certifies for IA | Per installment |
| Progress review | Bank + IA + Applicant | Quarterly |
| Disbursement coordination | Bank (loan) + IA (subsidy) | Per milestone |

### Bank Certificate Contents

**BANK PRACTICE** — Bank certificate typically includes:
- Applicant name and loan account number
- Certified expenditure amount
- Confirmation of no overdue installments
- Confirmation of collateral intact
- Date and authorized signature

## Common Delays in Grant Release

### Delay Causes

| Delay Cause | Description | Prevention | Status |
|-------------|-------------|------------|--------|
| Incomplete documentation | Missing UC, invoices, or certificates | Use [PMKS-007](PMKSY_MASTER/07_REQUIRED_DOCUMENTS.md) checklist | PROJECT ASSUMPTION |
| Bank certificate delay | Bank slow to issue certificate | Maintain regular bank relationship; follow up | BANK PRACTICE |
| Site inspection scheduling | IA / bank officials unavailable | Schedule inspection in advance | PROJECT ASSUMPTION |
| Expenditure mismatch | Claimed amount doesn't match verified amount | Maintain meticulous records | PROJECT ASSUMPTION |
| Query resolution | IA raises queries, applicant delays response | Monitor queries; respond within 7 days | PROJECT ASSUMPTION |
| Fund availability | State / central fund delays | Escalate if delayed beyond normal | REQUIRES RE-VERIFICATION |
| Compliance issues | Statutory lapses flagged | Maintain compliance tracker | OFFICIAL |

### Delay Mitigation

**ENGINEERING RECOMMENDATION** — To minimize grant release delays:
- Submit applications with complete, verified documentation
- Maintain pre-inspection readiness
- Follow up weekly post-submission
- Keep IA nodal officer informed of progress
- Maintain bank relationship for prompt certification

## Common Grant Rejection Causes

### Rejection Reasons

| Reason | Description | Prevention | Status |
|--------|-------------|------------|--------|
| Inflated expenditure | Claimed costs higher than actual / market | Use actual invoices; avoid inflated estimates | PROJECT ASSUMPTION |
| Ineligible costs included | Subsidy claimed on ineligible items | Strict cost classification per [PMKS-006](PMKSY_MASTER/06_ELIGIBLE_PROJECT_COST.md) | REQUIRES RE-VERIFICATION |
| Missing documents | Incomplete application package | Use [PMKS-007](PMKSY_MASTER/07_REQUIRED_DOCUMENTS.md) checklist | PROJECT ASSUMPTION |
| No physical progress | Claim without corresponding work | Ensure site ready for inspection | PROJECT ASSUMPTION |
| Bank dues | Applicant's loan account irregular | Maintain loan regularity | BANK PRACTICE |
| Compliance failure | FSSAI, environmental, or other violations | Maintain compliance tracker | OFFICIAL |
| Duplicate claiming | Same expense claimed in earlier release | Maintain release history | PROJECT ASSUMPTION |
| Beyond sanctioned timeline | Release requested after project completion deadline | Request timeline extension if needed | REQUIRES RE-VERIFICATION |
| Unauthorized changes | Project scope changed without IA approval | Seek IA approval for changes | OFFICIAL |

### Rejection Handling

**PROJECT ASSUMPTION** — If a grant release application is rejected:
1. Obtain written reasons from IA
2. Address deficiencies promptly
3. Resubmit with corrections
4. Escalate if unjustified (see [PMKS-009](PMKSY_MASTER/09_IMPLEMENTING_AGENCIES.md) escalation path)

## Subsidy Calculation Methodology

### Calculation Basis

**PROJECT ASSUMPTION** — PMKSY subsidy is calculated as:

```
Subsidy Amount = Eligible Project Cost (EPC) × Subsidy Rate %
```

### Factors Affecting Calculation

| Factor | Impact | Status |
|--------|--------|--------|
| EPC (sanctioned) | Base for calculation | OFFICIAL |
| Subsidy rate | Varies by area type (General / Difficult) | REQUIRES RE-VERIFICATION |
| Maximum subsidy ceiling | Caps total subsidy | REQUIRES RE-VERIFICATION |
| Promoter contribution | Does not affect subsidy amount | OFFICIAL |
| Bank loan | Does not affect subsidy amount | OFFICIAL |

### Release Calculation at Each Stage

**PROJECT ASSUMPTION** — At each release stage:

| Stage | Calculation | Typical Cumulative Release |
|-------|-------------|---------------------------|
| First | % of EPC × eligible expenditure incurred | 30-50% |
| Second | % of EPC × eligible expenditure incurred | 70-80% |
| Final | Balance of sanctioned subsidy | 100% |

**REQUIRES RE-VERIFICATION** — Exact percentages and release stages must be confirmed from current PMKSY guidelines and sanction letter.
