# Project Appraisal

## Overview

This document defines the complete project appraisal framework for PMKSY chocolate manufacturing projects in Andhra Pradesh. It covers technical appraisal, financial appraisal, bank appraisal methodologies, the interface between PMKSY subsidy appraisal and bank term loan appraisal, and common appraisal pitfalls.

**OFFICIAL** — PMKSY projects undergo multi-layered appraisal before sanction. Appraisal ensures that only technically feasible, financially viable, and eligible projects receive support.

> **SCOPE BOUNDARY**: This document addresses **project appraisal methodology and criteria**. It does **not** address:
> - Eligibility criteria (see [PMKS-002: Project Eligibility](PMKSY_MASTER/02_PROJECT_ELIGIBILITY.md))
> - Cost eligibility classification (see [PMKS-006: Eligible Project Cost](PMKSY_MASTER/06_ELIGIBLE_PROJECT_COST.md))
> - Document requirements (see [PMKS-007: Required Documents](PMKSY_MASTER/07_REQUIRED_DOCUMENTS.md))
> - Application workflow steps (see [PMKS-008: Application Workflow](PMKSY_MASTER/08_APPLICATION_WORKFLOW.md))
> - Agency roles (see [PMKS-009: Implementing Agencies](PMKSY_MASTER/09_IMPLEMENTING_AGENCIES.md))

## Appraisal Framework Overview

### PMKSY Appraisal vs. Bank Appraisal

**PROJECT ASSUMPTION** — PMKSY chocolate manufacturing projects undergo **two parallel but independent appraisals**:

| Appraisal | Purpose | Conducted By | Basis |
|-----------|---------|--------------|-------|
| **PMKSY Appraisal** | Verify eligibility, technical feasibility, financial viability for subsidy | Implementing Agency (IA) / MoFPI | PMKSY guidelines |
| **Bank Appraisal** | Verify bankability, collateral adequacy, creditworthiness for term loan | Scheduled Commercial Bank | Bank norms, RBI guidelines |

**BANK PRACTICE** — Banks appraise projects at 100% of Total Project Cost (TPC) without deducting projected PMKSY subsidy. This ensures promoter contribution is calculated on full cost.

**PROJECT ASSUMPTION** — Both appraisals must be favorable for project success. Discrepancies between the two are resolved through applicant revision or negotiated compromise.

### Appraisal Sequence

```
PROJECT CONCEPTION
        ↓
DPR PREPARATION
        ↓
┌─────────────────────────────────────────┐
│  PARALLEL APPRAISALS                     │
│  ┌──────────────┐    ┌──────────────┐  │
│  │ Bank Appraisal│    │ PMKSY Appraisal│ │
│  │ (Bank)        │    │ (IA / MoFPI)  │  │
│  └──────────────┘    └──────────────┘  │
└─────────────────────────────────────────┘
        ↓                       ↓
   BANK SANCTION          PMKSY SANCTION
        ↓                       ↓
        └───────────┬───────────┘
                    ↓
            IMPLEMENTATION
                    ↓
            GRANT RELEASE
```

## Technical Appraisal

### Purpose

Technical appraisal verifies that the proposed chocolate manufacturing project is technically feasible, uses appropriate technology, and can achieve the stated production objectives.

### Technical Appraisal Conducted By

| Agency | Role | Status |
|--------|------|--------|
| State Nodal Agency | Primary technical appraisal | REQUIRES RE-VERIFICATION |
| MoFPI | Review for large / innovative projects | REQUIRES RE-VERIFICATION |
| Bank | Independent technical opinion (may appoint consultant) | BANK PRACTICE |
| CFTRI / IIFPT | Advisory support (if consulted) | PROJECT ASSUMPTION |

### Technical Appraisal Criteria

#### Process Design

| Criterion | Requirement | Status |
|-----------|-------------|--------|
| Process flow adequacy | Complete process from cocoa beans to finished chocolate | ENGINEERING RECOMMENDATION |
| Technology maturity | Commercially proven technology | PROJECT ASSUMPTION |
| Process efficiency | Industry-standard yields and recovery | PROJECT ASSUMPTION |
| Quality assurance | HACCP-compatible design | PROJECT ASSUMPTION |
| Scalability | Future expansion capability | PROJECT ASSUMPTION |

**ENGINEERING RECOMMENDATION** — Process design should follow standard chocolate manufacturing sequence: roasting → winnowing → grinding → refining → conching → tempering → moulding → cooling → packaging.

#### Equipment Selection

| Criterion | Requirement | Status |
|-----------|-------------|--------|
| Equipment adequacy | Equipment rated for stated capacity | PROJECT ASSUMPTION |
| OEM reputation | Established manufacturers preferred | ENGINEERING RECOMMENDATION |
| Spare parts availability | Local / regional availability | ENGINEERING RECOMMENDATION |
| Maintenance capability | Service support in Andhra Pradesh | ENGINEERING RECOMMENDATION |
| Energy efficiency | Industry-standard power consumption | PROJECT ASSUMPTION |

**PROJECT ASSUMPTION** — For a ₹2 crore chocolate unit, typical equipment includes roasting machine, winnowing machine, ball mill/refiner, conche, tempering machine, moulding line, cooling tunnel, and packaging machine.

#### Capacity and Scale

| Criterion | Requirement | Status |
|-----------|-------------|--------|
| Installed capacity | Aligned with market demand | PROJECT ASSUMPTION |
| Utilization ratio | Realistic production schedule | PROJECT ASSUMPTION |
| Expansion headroom | 25-30% excess capacity preferred | ENGINEERING RECOMMENDATION |
| Technology capacity | Equipment-rated capacity vs. design capacity | PROJECT ASSUMPTION |

**REQUIRES RE-VERIFICATION** — Confirm minimum capacity thresholds for PMKSY Component 2 (food processing).

#### Civil and Infrastructure

| Criterion | Requirement | Status |
|-----------|-------------|--------|
| Built-up area adequacy | Meets zoning, safety, and operational needs | PROJECT ASSUMPTION |
| Structural design | Seismic Zone III / IV compliance (AP) | PROJECT ASSUMPTION |
| Utility capacity | Power, water, steam, compressed air | PROJECT ASSUMPTION |
| Effluent treatment | ETP / ZLD compliance | OFFICIAL — under environmental rules |
| Fire safety | Fire NOC compliance | OFFICIAL — state Fire Services Act |

#### Quality Assurance

| Criterion | Requirement | Status |
|-----------|-------------|--------|
| QC laboratory | FSSAI-compliance testing capability | OFFICIAL — FSSAI requirement |
| In-process controls | Critical control points identified | PROJECT ASSUMPTION |
| Testing protocols | Standard methods for chocolate parameters | PROJECT ASSUMPTION |
| Storage conditions | Temperature/humidity control | ENGINEERING RECOMMENDATION |

### Technical Appraisal Report Contents

**PROJECT ASSUMPTION** — A typical technical appraisal report includes:

| Section | Content |
|---------|---------|
| Executive summary | Overall technical feasibility assessment |
| Process description | Detailed process flow with P&ID |
| Equipment list | Itemized with specifications and quotations |
| Capacity analysis | Installed vs. utilized capacity |
| Civil works | Layout plan, structural adequacy |
| Utilities | Power, water, effluent, fire systems |
| Quality assurance | QC lab, testing protocols |
| Environmental aspects | ETP, emissions, waste management |
| Risk assessment | Technical risks and mitigations |
| Recommendations | Approval with conditions / revision required / rejection |

### Technical Appraisal Decision

| Decision | Criteria |
|----------|-----------|
| **APPROVED** | All technical criteria met; minor conditions acceptable |
| **APPROVED WITH CONDITIONS** | Technical viability confirmed; specific conditions to be fulfilled before / during implementation |
| **REVISION REQUIRED** | Technical gaps identified; applicant must revise and resubmit |
| **REJECTED** | Fundamental technical infeasibility |

## Financial Appraisal

### Purpose

Financial appraisal verifies that the project is financially viable, costs are reasonable, and the financial structure is sustainable.

### Financial Appraisal Conducted By

| Agency | Role | Status |
|--------|------|--------|
| State Nodal Agency | PMKSY financial appraisal | REQUIRES RE-VERIFICATION |
| Bank | Term loan financial appraisal | BANK PRACTICE |
| MoFPI | Review for large / high-subsidy projects | REQUIRES RE-VERIFICATION |

### Financial Appraisal Criteria

#### Cost Reasonableness

| Criterion | Requirement | Status |
|-----------|-------------|--------|
| Itemized cost breakup | Component-wise cost with vendor quotes | PROJECT ASSUMPTION |
| Market benchmark | Costs compared to industry norms | PROJECT ASSUMPTION |
| Vendor quotations | At least 3 quotes for major equipment | PROJECT ASSUMPTION |
| Civil cost norms | Per sq.m rates within market range | PROJECT ASSUMPTION |
| Contingency adequacy | 5-10% of capital cost | PROJECT ASSUMPTION |

**ENGINEERING RECOMMENDATION** — Provide at least 3 vendor quotations for equipment costing >₹10 lakh. For smaller items, market rate verification is acceptable.

#### Means of Finance

| Criterion | Requirement | Status |
|-----------|-------------|--------|
| Promoter contribution | Minimum as per scheme norms | REQUIRES RE-VERIFICATION |
| Bank term loan | Realistic quantum based on appraisal | BANK PRACTICE |
| PMKSY subsidy | As per applicable rates | REQUIRES RE-VERIFICATION |
| Other sources | Clearly identified and committed | PROJECT ASSUMPTION |
| Gap financing | Identified if TPC > means of finance | PROJECT ASSUMPTION |

**PROJECT ASSUMPTION** — Typical means of finance for ₹2 crore project:

| Source | Amount (₹ Cr) | Status |
|--------|--------------|--------|
| Promoter contribution | 0.50 (25%) | PROJECT ASSUMPTION |
| Bank term loan | 0.50 - 1.00 | PROJECT ASSUMPTION |
| PMKSY subsidy | [REQUIRES RE-VERIFICATION] | REQUIRES RE-VERIFICATION |
| Own funds / others | Balance | PROJECT ASSUMPTION |

**REQUIRES RE-VERIFICATION** — Confirm exact promoter contribution percentage and maximum project cost limits under current PMKSY.

#### Financial Viability

| Criterion | Requirement | Typical Threshold | Status |
|-----------|-------------|-------------------|--------|
| Debt Service Coverage Ratio (DSCR) | Ability to service loan | >1.5x | PROJECT ASSUMPTION |
| Internal Rate of Return (IRR) | Project profitability | >15% | PROJECT ASSUMPTION |
| Payback period | Capital recovery | <7 years | PROJECT ASSUMPTION |
| Break-even capacity | Minimum production for breakeven | <70% utilization | PROJECT ASSUMPTION |
| Working capital adequacy | WC gap coverage | 15-20% of TPC | PROJECT ASSUMPTION |

**ENGINEERING RECOMMENDATION** — Prepare financial projections for 5 years with conservative assumptions. Show sensitivity analysis for key variables (sales volume, realization, raw material cost).

#### Promoter Financial Credibility

| Criterion | Requirement | Status |
|-----------|-------------|--------|
| Net worth | Adequate to support contribution | REQUIRES RE-VERIFICATION |
| Source of funds | Documentary evidence for own funds | OFFICIAL |
| Track record | Prior project execution / repayment | PROJECT ASSUMPTION |
| Creditworthiness | No adverse CIBIL / bank defaulter status | BANK PRACTICE |

## Bank Appraisal

### Purpose

Bank appraisal determines whether the project is bankable — i.e., whether the bank can safely sanction and monitor a term loan.

### Bank Appraisal Process

**BANK PRACTICE** — Typical bank appraisal process for ₹2 crore food processing project:

```
1. Loan application receipt
        ↓
2. KYC and initial screening
        ↓
3. Technical appraisal (in-house or consultant)
        ↓
4. Financial appraisal (in-house)
        ↓
5. Collateral assessment (valuer)
        ↓
6. Legal opinion (title, entity)
        ↓
7. Site visit
        ↓
8. Appraisal report preparation
        ↓
9. Sanction committee approval
        ↓
10. Sanction letter issuance
```

### Bank Appraisal Criteria

#### Technical Appraisal (Bank)

| Criterion | Bank Focus | Status |
|-----------|------------|--------|
| Location | Site accessibility, infrastructure | BANK PRACTICE |
| Technology | Proven, locally supportable | BANK PRACTICE |
| Capacity | Realistic vs. market demand | BANK PRACTICE |
| Management | Promoter capability | BANK PRACTICE |
| Risk assessment | Identifiable and mitigable | BANK PRACTICE |

#### Financial Appraisal (Bank)

| Criterion | Bank Focus | Typical Requirement |
|-----------|------------|---------------------|
| DSCR | Debt servicing capacity | >1.3-1.5x |
| IRR | Project profitability | >12-15% |
| Promoter contribution | Skin in the game | Typically 25-35% |
| Collateral coverage | Loan-to-value ratio | Typically 60-70% |
| Working capital | WC gap funding | Separate or part of TL |

#### Collateral Assessment

| Criterion | Bank Requirement | Status |
|-----------|------------------|--------|
| Land valuation | Market value by approved valuer | BANK PRACTICE |
| Building valuation | Post-construction value | BANK PRACTICE |
| Machinery valuation | Market / fit value | BANK PRACTICE |
| Loan-to-value ratio | 60-70% of collateral value | BANK PRACTICE |
| Margin money | Promoter contribution | Typically 30-40% |

### Bank Appraisal Report

**BANK PRACTICE** — Bank appraisal report typically contains:

| Section | Content |
|---------|---------|
| Executive summary | Appraisal recommendation |
| Project description | Overview of chocolate manufacturing unit |
| Market assessment | Demand, competition, pricing |
| Technical feasibility | Process, equipment, capacity |
| Financial analysis | Projected P&L, cash flows, balance sheet |
| Means of finance | Capital structure, sources |
| Risk analysis | Key risks and mitigants |
| Collateral security | Security structure, valuation |
| Terms and conditions | Loan terms, covenants, guarantees |
| Recommendation | Approve / approve with conditions / reject |

## PMKSY Appraisal vs. Bank Appraisal: Interface

### Key Differences

| Dimension | PMKSY Appraisal | Bank Appraisal |
|-----------|-----------------|----------------|
| Primary objective | Eligibility and subsidy justification | Credit risk and loan recovery |
| Cost basis | Eligible Project Cost (EPC) | Total Project Cost (TPC) |
| Subsidy treatment | EPC × subsidy rate | Not considered for loan quantum |
| Collateral focus | Not primary | Primary security for loan |
| Promoter focus | Eligibility and capability | Creditworthiness and net worth |
| Timeline | Linked to IA / MoFPI process | Bank's internal timelines |
| Approval authority | IA / MoFPI | Bank sanction committee |

### Coordination Mechanisms

**BANK PRACTICE** — Banks and IAs coordinate through:

| Mechanism | Description | Frequency |
|-----------|-------------|-----------|
| Joint site inspection | Both verify physical progress | Per subsidy installment |
| Expenditure certification | Bank certifies before IA releases subsidy | Per installment |
| Progress review meetings | Quarterly or milestone-based | Quarterly |
| Grievance escalation | Joint resolution of disputes | As needed |

### Subsidy-Bank Interface Issues

| Issue | Typical Resolution | Status |
|-------|--------------------|--------|
| Bank appraises at 100% TPC; PMKSY subsidy reduces promoter burden | Bank does not reduce loan quantum | BANK PRACTICE |
| Discrepancy in cost estimates | Applicant provides market verification | PROJECT ASSUMPTION |
| Delay in PMKSY sanction affecting bank timeline | Bank may sanction with MoFPI in-principle condition | REQUIRES RE-VERIFICATION |
| Bank requires additional guarantees beyond PMKSY requirements | Negotiated between applicant and bank | PROJECT ASSUMPTION |

**REQUIRES RE-VERIFICATION** — Confirm whether all banks accept MoFPI in-principle approval before final sanction, or if specific banks require independent appraisal without PMKSY linkage.

## Common Appraisal Pitfalls and Rejection Reasons

### Technical Appraisal Pitfalls

| Pitfall | Description | Prevention |
|---------|-------------|------------|
| Inadequate process design | Unproven or unsuitable technology | Use proven OEMs; get consultant review |
| Overstated capacity | Equipment rating vs. realistic output | Realistic capacity planning with 20% buffer |
| Missing environmental compliance | ETP not designed / approved | Engage environmental consultant early |
| Inadequate QC lab | FSSAI compliance gaps | Design lab per FSSAI requirements |
| Poor layout | Inefficient material flow | Follow process flow in layout design |

### Financial Appraisal Pitfalls

| Pitfall | Description | Prevention |
|---------|-------------|------------|
| Inflated costs | Overstated to maximize subsidy | Use market rates; provide vendor quotes |
| Unrealistic projections | Overstated sales, understated costs | Conservative assumptions; sensitivity analysis |
| Inadequate promoter contribution | Minimum contribution not met | Ensure source of funds documented |
| Missing working capital | Subsidy assumed to cover WC | Separate WC arrangement |
| Weak DSCR | Low debt service coverage | Restructure finance or reduce loan |

### PMKSY-Specific Appraisal Pitfalls

| Pitfall | Description | Prevention |
|---------|-------------|------------|
| Ineligible cost included | Land, WC, or other ineligible items in EPC | Strict cost classification per PMKS-006 |
| Missing documents | Incomplete application package | Use PMKS-007 checklist |
| Eligibility misunderstanding | Applicant, entity, or project not meeting criteria | Self-assess against PMKS-002, PMKS-003, PMKS-004 |
| Land issues | Title disputes, land-use problems | Complete land eligibility verification (PMKS-005) |
| Wrong component | Applying under wrong PMKSY component | Confirm component applicability |
