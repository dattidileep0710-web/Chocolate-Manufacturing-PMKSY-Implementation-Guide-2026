# Implementing Agencies

## Overview

This document defines the ecosystem of agencies involved in PMKSY implementation for a chocolate manufacturing project in Andhra Pradesh. It clarifies roles, responsibilities, authority boundaries, interaction workflows, and escalation paths so that applicants, banks, and consultants understand who does what at each stage.

**OFFICIAL** — PMKSY is implemented through a multi-agency framework involving the Union Government, state governments, financial institutions, and regulatory authorities. Each agency has defined roles and boundaries.

**PROJECT ASSUMPTION** — For a ₹2 crore chocolate manufacturing unit in Andhra Pradesh, the primary agencies are MoFPI (nodal ministry), a designated State Nodal Agency, NABARD (financial partner), scheduled commercial banks (term loan), and statutory departments (compliance).

> **SCOPE BOUNDARY**: This document addresses **agency roles and interactions**. It does **not** address eligibility criteria, document checklists, cost classification, or application forms — those are covered in PMKS-002 through PMKS-008.

## Agency Ecosystem Summary

### PMKSY Implementation Architecture

**PROJECT ASSUMPTION** — The PMKSY implementation architecture for food processing projects in Andhra Pradesh follows this hierarchy:

```
MoFPI (Nodal Ministry)
    ↓
State Nodal Agency / Implementing Agency
    ↓
District-level Facilitation / District Industries Centre
    ↓
Applicant / Bank / Statutory Departments
```

| Agency Type | Examples | Role Category |

| Nodal Ministry | MoFPI | Policy, fund allocation, oversight |
| State Nodal Agency | AP State Food Processing Society / AP Food Processing Mission | State-level coordination, first-level appraisal |

| Lending Institution | Scheduled Commercial Banks | Term loan sanction, disbursement, monitoring |
| District Authority | District Industries Centre (DIC) | District-level facilitation, MSME support |

## Ministry of Food Processing Industries (MoFPI)

### Role and Mandate

**OFFICIAL** — MoFPI is the nodal ministry for PMKSY. It is responsible for:
- Scheme design, policy formulation, and amendment
- Budget allocation and fund release to implementing agencies

## NABARD (National Bank for Agriculture and Rural Development)

- Channelizing funds to banks and financial institutions

## Applicant Interaction Workflow

### Applicant Journey Through Agencies

**PROJECT ASSUMPTION** — An applicant for a ₹2 crore chocolate manufacturing unit in Andhra Pradesh interacts with agencies as follows:

```
1. Applicant → Bank: Pre-feasibility, term loan application
2. Applicant → DIC: Awareness, MSME registration facilitation
3. Applicant → State Nodal Agency: PMKSY application submission, queries, sanctions
4. Applicant → MoFPI: Rarely direct; only if project escalates to Ministry level
5. Applicant → NABARD: Rarely direct; typically through bank
```

## Escalation Path

### Applicant Escalation Hierarchy

**PROJECT ASSUMPTION** — If issues arise, applicants should escalate in this order:

```
Level 1: State Nodal Agency nodal officer
    ↓ (if unresolved in 15 days)
Level 2: State Nodal Agency CEO / Managing Director
    ↓ (if unresolved in 15 days)
Level 3: MoFPI PMKSY Division
    ↓ (if unresolved in 30 days)
Level 4: Minister of State / Cabinet Minister (MoFPI)
```

### Bank Escalation Hierarchy

**BANK PRACTICE** — Banks escalate through:

```
Level 1: Bank's MSME / food processing desk
    ↓
Level 2: Bank's regional / zonal office
    ↓
Level 3: NABARD regional office
    ↓
Level 4: NABARD head office / MoFPI
```
